# React-marquee
===========
The marquee tag in html is obsolete and not well supported by modern browsers. This tool makes use of CSS3 to create a replacement.

## Installation
1. Node
2. NPM

## Make it run
1. npm install
2. npm start
3. open http://localhost:3005
